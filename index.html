DG Consultoria

Sistema web desenvolvido para a empresa DG Consultoria, especializada em segurança do trabalho.O projeto é uma plataforma de venda de cursos online, onde os clientes podem adquirir treinamentos relacionados à segurança do trabalho. O sistema inclui funcionalidades como catálogo de cursos, processamento de pagamentos, onde a empresa recebe o email do pagante para enviar o curso, e uma área administrativa para gerenciar os cursos e os clientes. O objetivo do projeto é fornecer uma solução eficiente e fácil de usar para a venda de cursos online, atendendo às necessidades da empresa e proporcionando uma experiência positiva para os clientes.
